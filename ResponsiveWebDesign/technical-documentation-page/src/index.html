Introduction

The present project1 was created by gpm222 based on the tutorial Intro to React3 by the ReactJS team.

Tic-tac-toe (American English), noughts and crosses (Commonwealth English), or Xs and Os (Irish English) is a paper-and-pencil game for two players who take turns marking the spaces in a three-by-three grid with X or O. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner. It is a solved game, with a forced draw assuming best play from both players.4

Requirements

It is necessary to have both git and npm installed.

Git can be installed by following the instructions on the git official page.5

Npm can be installed by following the instructions on the npm official page.6

Setup

Now it is necessary to install the tic-tac-toe game, this will be done by following the next steps:

  • First you have to clone the repository, which contains the tic-tac-toe project, by running the following command in a terminal:

    git clone https://github.com/gpm22/BookExercises.git

  • Then you have to enter the folder which contains the ti-tac-toe project. You will do it by running the following command.

    cd BookExercises/React/IntroToReact/tic-tac-toe

  • Now you have to install all the necessary components needed to put an React app online. This is done by running the following command inside the folder containing the project:

    npm install

  • Now you just have to start the app to be able to play. This is done by running the following command inside the folder containing the project:

    npm start

  • If the entire setup was done correctly, a message similar to the following will appear on the terminal:

    Compiled successfully! You can now view tic-tac-toe in the browser. Local: http://localhost:3000 On Your Network: http://xxx.xxx.x.xx:3000 Note that the development build is not optimized. To create a production build, use npm run build.

Now you can play the game by acessing the url http://localhost:3000 in your favorite browser. Have fun!

How to Play

The objective of the game is to make a linear sequence of Xs or Os. The sequence que be a row, a colum, or a diagonal.

The game is played in alternate turns, where each player takes their turn.

The first player is the one who plays with X and the second player is the one who plays with O.

Every move made is saved in a list on the left side of the board, so it is possible to go back to any previous play.

  • The move is indicated by the order in which it occured and show what was done.

  • Ex: Go to move #1 - X in position (1,2)
